Welcome to XALOC automated data processor

Select your main working directory using the "Change directory" button in the up-right corner, you tipically select a directory corresponding to a specific beamtime such as /beamlines/bl13/commissioning/YYYYMMDD (If you haven't done it yet from the terminal)

Once you have selected the directory select the dataset you want to work with from the list to start working.
The color of each dataset indicates its current stage which can be:
- Images to mtz (red) Stage in which the data images haven't been converted to .mtz yet
- Phasing (orange) Stage in which the phasing problem is yet to be solved
- Refinement (green) Stage where there is an accepted solution

Otherwise select a stage to list only the datasets which are in the selected stage.
